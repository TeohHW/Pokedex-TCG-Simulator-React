# React Workspace Template

A lightweight React starter powered by Vite.

## Scripts

- `npm.cmd install` installs dependencies on Windows PowerShell.
- `npm.cmd run dev` starts the local development server.
- `npm.cmd run build` creates a production build.
- `npm.cmd run preview` serves the production build locally.
- `npm.cmd run lint` runs ESLint.

## Start Editing

The app entry point is `src/main.jsx`.

The main component is `src/App.jsx`.

Global styles live in `src/styles.css`.
